# Software
I am a proficient Stata user and use Stata for research, programming, and teaching. I am comfortable using L<sup style="font-size: 0.7em; vertical-align: 0.2em;">A</sup>T<sub style="font-size: 0.7em; vertical-align: -0.2em;">E</sub>X and have intermediate understanding of R software.

### Missing data
We recently published a Stata software package with a description of the package and applications in the [Stata Journal](https://journals.sagepub.com/doi/10.1177/1536867X251398605).

[`mi impute from`](https://ideas.repec.org/c/boc/bocode/s459378.html), is a package to impute binary, categorical, and continious variables from external data sources using logistic, multinomial logistic, and quantile regression, respectively. It is avaialble to download from the Statistical Software Component Archive at Boston College (`ssc`).

Other packages that are related to `mi impute from` and that I contributed to and are available to download:

`mi impute cqi`, a package to impute categorical missing data using conditional quantiles. It can be downloaded into Stata with `net describe http://www.stats4life.se/stata/mi\_impute\_cqi`

`mi impute quantile`, a package to impute continuous missing values using quantile regression. It can be used with `net describe http://www.stats4life.se/stata/mi\_impute\_quantile`
